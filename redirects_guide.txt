# Redirect-Konzept (301) für bestehende Domains

# Ziel:
#  - argo-wohnen-leipzig.de   -> https://www.argo-wohnen.de/standorte/leipzig/
#  - argo-wohnen-magdeburg.de -> https://www.argo-wohnen.de/standorte/magdeburg/
#  - argo-wohnen-dresden.de   -> https://www.argo-wohnen.de/standorte/dresden/

# Wichtig:
#  - 301 (permanent) verwenden
#  - Query-Strings (gclid/utm) beibehalten (Apache: QSA / Nginx: $args)
#  - Optional: Datenschutz/Impressum sauber auf neue Seiten mappen

## Apache / .htaccess (Beispiel: Leipzig-Domain)
# RewriteEngine On
# RewriteCond %{HTTP_HOST} ^(www\.)?argo-wohnen-leipzig\.de$ [NC]
# RewriteRule ^datenschutz/?$ https://www.argo-wohnen.de/datenschutz/ [R=301,L,QSA]
# RewriteRule ^impressum/?$ https://www.argo-wohnen.de/impressum/ [R=301,L,QSA]
# RewriteRule ^(.*)$ https://www.argo-wohnen.de/standorte/leipzig/ [R=301,L,QSA]
